# 🀄 麻将游戏 (mjGame)

四人局域网联机麻将网页游戏，支持 AI 机器人自动对局。

## 规则说明

- **牌组**: 136 张（万/筒/条各 36 张 + 东南西北风 16 张 + 中发白箭 12 张）
- **操作**: 只支持**碰**和**杠**，不支持吃
- **赖子**: **红中**作为赖子（百搭牌），可替代任意牌组成胡牌牌型
- **杠的类型**: 暗杠（手中 4 张相同）、明杠（别人打出 + 手中 3 张）、加杠（已碰后再摸第 4 张）
- **胡牌条件**: 4 组面子（刻子/顺子）+ 1 对雀头，允许赖子替代缺牌
- **流局**: 牌墙耗尽时无人胡牌则为流局

## 快速开始

### 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0
- 现代浏览器（Chrome / Firefox / Safari / Edge）

### 安装与启动

```bash
# 1. 克隆仓库
git clone https://github.com/MountainRs/mjGame.git
cd mjGame

# 2. 安装依赖
npm install

# 3. 启动服务器
npm start
```

启动后会看到:

```
========================================
  🀄 麻将游戏服务器已启动！
  本机访问: http://localhost:3000
  局域网访问: http://192.168.x.x:3000
========================================
```

### 游戏流程

1. **创建/加入房间**
   - 打开浏览器访问 `http://localhost:3000`（本机）或 `http://192.168.x.x:3000`（局域网）
   - 输入昵称，点击「创建房间」或输入房间号点击「加入房间」
   - 房间号可通过「复制链接」按钮分享给其他玩家

2. **添加玩家**
   - 局域网内其他玩家通过房间号或链接加入
   - 不足 4 人时，点击「添加机器人」用 AI 填充空位

3. **开始游戏**
   - 4 人到齐后，点击「开始游戏」
   - 庄家随机确定，庄家先出牌

4. **操作方式**
   - **出牌**: 点击手中的牌打出
   - **碰/杠/胡**: 别人打出牌后，如果可以操作会弹出按钮
   - **自摸胡**: 自己摸牌后如果能胡，会弹出胡牌提示
   - **暗杠/加杠**: 自己回合如果有可杠的牌，会弹出杠牌提示
   - **跳过**: 点击「跳过」放弃当前可选操作，继续出牌

5. **游戏结束**
   - 有人胡牌或流局后，显示所有人的手牌
   - 点击「再来一局」重新开始

### 局域网联机

在同一 WiFi/局域网下的设备都可以访问:

1. 主机启动服务器 (`npm start`)
2. 其他设备浏览器输入局域网地址（控制台会显示 IP）
3. 分享链接格式: `http://<IP>:3000/?room=<房间号>`

## 项目结构

```
mjGame/
├── package.json           # 项目配置
├── server/
│   ├── index.js           # 服务入口 (Express + WebSocket)
│   ├── Game.js            # 游戏引擎核心
│   ├── huChecker.js       # 胡牌检测算法（含赖子）
│   ├── ai.js              # AI 机器人逻辑
│   └── tiles.js           # 牌面定义与工具函数
├── public/
│   ├── index.html         # 单页应用 HTML
│   ├── style.css          # 样式
│   └── app.js             # 前端逻辑
└── test/
    └── test.js            # 测试套件
```

## 技术栈

| 层 | 技术 |
|---|---|

| 通信 | WebSocket (JSON 消息协议) |
